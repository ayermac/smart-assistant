# 产品定义

## 目标

`smart-assistant` 是一个面向个人知识管理的智能助手，帮助用户在本地知识库、个人记忆和日常任务之间建立统一的交互入口。

## 目标用户

- 需要长期管理个人笔记、文档和任务的人
- 想要一个可控、可审计、默认本地运行的 AI 助手的人
- 希望先用 CLI 跑通 agent 工作流，再考虑扩展到别的入口的人

## 核心场景

- 询问本地笔记中的事实
- 让助手记住个人偏好和长期信息
- 让助手把复杂任务拆成步骤
- 让助手从知识库里查出相关片段并总结

## 用户流程

1. 启动 CLI
2. 选择或恢复会话
3. 输入问题或任务
4. 助手判断是否需要记忆、检索或规划
5. 输出结果或执行工具
6. 将必要结果写回会话和长期存储

## MVP

- CLI 对话
- 单 agent loop
- 本地会话保存
- 个人记忆的写入和检索
- Markdown/text 知识库检索
- 简单任务规划

## 非目标

- 不做通用聊天产品
- 不做多租户 SaaS
- 不做默认云同步
- 不做复杂多模态输入
- 第一版不做浏览器自动化或桌面自动化

## 成功标准

- 用户能在 CLI 里持续对话
- 助手能记住明确写入的长期信息
- 助手能从本地知识库找到相关资料
- 助手能把任务拆解成可执行步骤
- 每个能力都能用固定测试用例验证

## 第一版成功标准

- 能在本地跑起来
- 能恢复会话
- 能记住一条偏好并在后续回答中复用
- 能从本地 Markdown 知识库检索到相关片段
- 能输出一个可执行的任务计划
